# Sponsors Images

Upload sponsor logo images here.

## Guidelines:
- Use PNG or SVG format
- Recommended size: 256x256 pixels
- Maximum file size: 500KB
- File naming: `sponsor-name.png` or `sponsor-name.svg`

## Usage:
After uploading, use the URL: `/sponsors/your-logo.png`

Example in Admin Panel:
```
https://yourdomain.com/sponsors/google-logo.png
```
